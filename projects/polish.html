Nail Polish Twitter Bot:

Code | Site

This Twitter bot randomly picks a nail polish color from a JSON file and tweets the brand, color name and which collection that color is from. The JSON data was collected by scraping the brandsâ€™ sites.

Follow @AnotherCoat

Challenge: What to tweet?

Solution: I had to figure out what I wanted my bot to tweet. I ended up deciding to do nail polish colors because there are a lot of fun names out there!


Challenge: Scrape the internet for full lists of color names.

Solution: I Googled around to find pages with comprehensive lists of colors for three popular nail polish brands: OPI, Essie and Butter London. Once I found what I was looking for, I inspected the HTML to see how the data was formatted. I scraped the sites to create an array of HTML items by typing var brand = document.getElementsByClassName('class') into the console.


Challenge: Format data effectively to be randomly grabbed and tweeted.

Solution: Once I had an array of HTML elements, I figured out what to search for to find the data I needed from each item in the array. From there I wrote a for loop to grab the brand, color and collection from each element, which I then added to a JavaScript object. I ended up with a JSON formatted to have collections stored as keys whose values were objects containing each color as a key with the brand as it's value. Ex:
{
"Sheers": {
"Blanc": "Essie"
}
}
Then I wrote a function to randomly select a collection, from which it would randomly select a color which it would format as a sentence, var tweet = color + ' by ' + brand + ' from the ' + collection + ' Collection.', and send as a tweet.
