Mismatch Web App:

Code | Site

Mismatch is a browser-based web app that visitors log into using their Instagram accounts. It communicates with the Instagram API to get a list of the visitor's followers as well as the accounts that user follows. This information is passed through a node server before being graphically displayed using CSS. Once logged in, the visitor can search other handles to find information on the specified user.


Challenge: Hide the user's access token.

Solution: When the app first went live, there was no server and the access token was passed to the app attached to the redirect URL. To keep this from being shared by anyone passing the link along, I stored the token as a JavaScript variable and cut the token out of the URL window.


Challenge: Display difference between mismatches.

Solution: I found the difference between a user's followers and the accounts that user follows by creating an array for each, comparing them and then creating two new arrays called 'Not Following' and 'Not Followed By.' At first, I displayed the data in two lists, but that wasn't aesthetically pleasing. Then I merged the array with the full list of followers and accounts the user follows, making sure to mark the mismatches graphically. Finally, I decided to display one master list of all of the users associated with an account using CSS to visualize the relationship between the user and their connections. This also proved helpful to an additional challenge, getting as much information as possible without exceeding the amount of calls to the API that Instagram allows.


Challenge: When a user searched for a different handle, their connections were still showing up under the searched user's heading.

Solution: After logging some information to the console, I saw that handle that was typed in was, indeed, getting searched. That led me to look to the bottom of the list, where I found the new search's connections appended. It was an easy fix to make sure the old information was cleared every time a new search was performed.
