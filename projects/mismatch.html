Mismatch Web App:

Code | Site

This is a browser-based web app that communicates with the Instagram API in order to get list of the visitor's followers and the accounts that user follows. This information is passed through a node server before being graphically displayed on the left hand side using CSS.

GOAL:

Build a web app using the Instagram API. As I built it, this goal morphed into building an app to show the difference between the accounts a user follows and what that user is followed by.

CHALLENGES:

Hide the user's access token.

Build my first node server.

Get as much information as possible without exceeding the amount of calls to the API that Instagram allows.
