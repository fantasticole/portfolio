Tic-Tac-Toe:

Code | Site

This is the first project I built from scratch for my application to Hacker School. It is a two-player Tic-Tac-Toe game that tracks how many wins each player has and alternates which player starts each game.

GOAL:

Build a tic-tac-toe game that demonstrates programming ability to gain admission to Hacker School Recurse Center.

CHALLENGES:

Check for the winner efficiently each time a move is made.

Stop play once there is a victor or stalemate.

Track the amount of wins each player has.

Alternate first player.
